What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface provided by web browsers that represents an HTML or XML document as a tree structure of objects. Each node in this tree corresponds to a part of the document, such as elements, attributes, and text. JavaScript can access, modify, and manipulate the content and structure of a web page via the DOM.

How do you select an element from the DOM?

One can select an element from the DOM using various methods provided by the document object in JavaScript. Common ways to select elements include:
1:let element = document.getElementById('myId')
2:let elements = document.getElementsByClassName('myClass')
3: let elements = document.getElementsByTagName('div')
4: let element = document.querySelector('.myClass')
5: let elements = document.querySelectorAll('.myClass');

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where one attach a single event listener to a parent element instead of adding event listeners to individual child elements. The event listener on the parent listens for events that bubble up from the child elements.

How do you manipulate an element's attributes and styles using the DOM?

1: document.querySelector('a').getAttribute('href')
2: document.querySelector('img').setAttribute('src', 'new-image.jpg')
3: document.querySelector('input').removeAttribute('disabled');
